## Назначение vault
Этот vault нужен для:
- визуализации связей проекта
- навигации по документам
- sprint-обзоров
- инженерных заметок
- карты системы разработки

## Правила
1. Источник истины по задачам — GitHub Project.
2. Источник истины по коду и документам — git.
3. Этот vault используется как визуальный слой и рабочая карта.
4. Важные решения должны дублироваться в docs/ и/или ADR.
